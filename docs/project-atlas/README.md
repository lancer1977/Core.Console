# Project Atlas

## Purpose

`Core.Console` provides console application building blocks for PolyhydraGames projects: dependency injection setup, page navigation, Spectre.Console rendering, and sample host wiring.

## Primary Validation

```bash
dotnet test PolyhydraGames.Core.Console.sln --configuration Release --verbosity minimal
```

## Package Surface

- Main project: `src/PolyhydraGames.Core.Console.csproj`
- Tests: `test/PolyhydraGames.Core.Console.Tests`
- Sample: `samples/SampleHost`
